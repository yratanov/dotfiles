Installation Tools on MacOS
==================

```
brew install alacritty neovim stow fzf rbenv ruby-build tmux tmuxinator gh koekeishiya/formulae/skhd
skhd --start-service
```

Installing configs
=====================
stow 2.4.1 required


```
stow --dotfiles --target=$HOME .
```


Installing on Arch
====================

```
./arch-install.sh
```
